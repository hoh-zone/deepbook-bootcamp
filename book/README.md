# DeepBook 开发权威指南

本目录是正式书稿与配套代码的工作区。

- `SUMMARY.md`：全书目录。
- `LEARNING_PATH.md`：从易到难的六阶阅读路线。
- `SOURCE_DEFINITION_ATLAS.md`：关键结构体、方法和事件的内嵌源码定义地图。
- `MOVE_READING_GUIDE.md`：高级 Move + DeepBook 的阅读方法。
- `STYLE_GUIDE.md`：章节写作规范。
- `chXX/README.md`：每章导读、源码地图、小节目录和章节检查清单。
- `chXX/NN-title.md`：每个编号小节的独立正文。
- `chXX/code/`：每章配套代码。

本书以 [MystenLabs/deepbookv3](https://github.com/MystenLabs/deepbookv3/tree/663edbf9c30d6c93100e6cd66936e1487a5dc9e0) 的 GitHub 源码为事实基础，围绕 DeepBookV3、DeepBook Margin、DeepBook Predict、SDK 集成、Indexer 和生产应用开发展开。写作目标是高级 Move 案例书 + DeepBook 专题书：先从产品直觉和最小交易入手，再进入源码细节，最后落到真实应用和生产交付。
